To-Do Lists and Task Manager

This project utilized my foundational knowledge of C# by ulitising classes, collections, loops, a local database (text files), logical if statements etc. in order to practise my skills and implement them in a project environment.

Feature Breakdown
Tasks

In the tasks section, a new task can be created. They enter a name, description, option for a deadline, image and text colour. Once the deadline is missed, the task is moved from the current tasks list into the missed tasks list.

The user has an option to remove any task they choose, even if have failed to complete it on time.

The user can also update the information inside of tasks to (e.g. update the due date) meet any changing requirements.

To-Do Lists

In To-Do lists, the user can create a whole new empty list of tasks and view it by selecting the list from the dropdown box below list create button. Although if the user wants to delete the new list there is an option to do that aswell, by pressing the "Remove" button.

The user can create tasks in this new list, but there are no deadlines. Instead it is ordered by priority as demonstrated in the GIF. The ones with the priority box checked are at the top of the list whereas the rest of the tasks are at the bottom of the list.
