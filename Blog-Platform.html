Blog Platform

In this Blog Platform project, I developed a confident understanding of backend web development and PHP as a whole.

Feature Breakdown
Standard Users

As a standard user, you have the option to make a post with comments or without comments using a tickbox. If it is unchecked, when the post is made there will be no link taking you to a page where you can make comments on a post.

In posts, there are many other things that can be included such as links to websites, an image, or a video. All 3 can also be displayed in the same post.

When a user wants to find a specific account to read the posts they made, there is a search bar where they can enter a query and find the profile they are looking for.

Each user has the option to like a post and also like the comments on the post, utilising database relationships so each user can only like a post 1 time.

On the homepage, posts can be sorted by most liked, most recent and most commented.

Moderators

Moderators can flag user accounts that they find suspicious, which then notifies the admin. This will be shown next.

Admins

In the admin dashboard, there is a page which can view the pages that the moderator flagged. They can then choose to dismiss the flagging or delete the account.

Admins can add context to every user post to increase clarity to other users on the context. Other pages on the admin dashboard include "latest posts", displaying the latest posts made on the site to monitor them.
