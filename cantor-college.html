Cantor College Website

Many of the pages in this web project are static as I am focusing on developing good practises (e.g. Indentation, correct tags, correct use of CSS). However, there is usage on the courses page.

Feature Breakdown

The NavBar contains links navigating to the different pages within the website. This one in paticular uses an embedded map with the location of the building, with the address in text to the right of it.

This page is connected to a database. It fetches the courses that the college provides and displays them on the page, along with the number of results currently showing at the top. The user has the ability to filter the courses to see their desired one, but as it was my first attempt I was not completely successful.

The rest of the website is more static webpages, reformatted to fit the style of content for that page. E.g. some will contain images, some will contain just text etc.
